# Portfolio Website - Codesoft Level 1 Task 1

A professional personal portfolio website built with HTML5, CSS3, and vanilla JavaScript to showcase skills, projects, and resume.

## 🎯 Project Overview

This is a single-page portfolio website designed to:
- Introduce yourself professionally
- Showcase your skills with visual progress indicators
- Display your best projects
- Provide contact information
- Build your personal brand

## ✨ Key Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Smooth Navigation**: Sticky header with smooth scrolling to sections
- **Hamburger Menu**: Mobile-friendly navigation menu for small screens
- **Animated Hero Section**: Eye-catching introduction with gradient background
- **Skills Display**: Visual progress bars showing skill proficiency
- **Projects Grid**: Showcase your best work with project cards
- **Resume Section**: Download button for your resume
- **Back-to-Top Button**: Easy navigation for long pages
- **Intersection Observer**: Smooth animations as you scroll down
- **Modern Design**: Gradient backgrounds, smooth transitions, hover effects

## 🚀 Quick Start

1. **Open the website**:
   - Simply open `index.html` in your web browser
   - Or deploy to GitHub Pages/Netlify/Vercel (see Deployment section)

2. **Customize with your information**:
   - Replace placeholder texts with your actual data
   - Update the skills and progress percentages
   - Add your own projects
   - Change colors in `style.css` (look for CSS variables in `:root`)

## 🛠️ Technologies Used

- **HTML5**: Semantic markup with proper structure
- **CSS3**: Flexbox, Grid, animations, CSS variables, media queries
- **JavaScript**: Vanilla JS (no frameworks), DOM manipulation, Intersection Observer API
- **Icons**: Font Awesome 6.4.0 (CDN)
- **Images**: Placeholder images (use your own)

## 📝 How to Customize

### 1. Update Your Information
Open `index.html` and replace:
- Your name in header and about section
- Profile image URL
- About section text
- Skills and proficiency levels
- Project descriptions and images
- Contact email and phone
- Social media links

### 2. Modify Colors
Open `style.css` and update CSS variables:
```css
:root {
    --primary-color: #667eea;      /* Change to your color */
    --secondary-color: #764ba2;    /* Accent color */
    /* ... other variables ... */
}
```

### 3. Add Your Resume
- Replace `resume.pdf` with your actual resume file
- Update the resume download link in HTML

### 4. Add Images
- Create an `images` folder
- Add your profile picture, project screenshots
- Update image paths in HTML

## 🚢 Deployment Options

### Option 1: GitHub Pages (Free & Easy)
1. Create a GitHub repository named `your-username.github.io`
2. Push this project to the repository
3. Your site is live at `https://your-username.github.io`

### Option 2: Netlify (Free)
1. Go to [netlify.com](https://netlify.com)
2. Click "New site from Git"
3. Connect your GitHub repository
4. Deploy automatically on every push

### Option 3: Vercel (Free)
1. Go to [vercel.com](https://vercel.com)
2. Import your GitHub repository
3. One-click deployment

### Option 4: 000webhost (Free)
1. Sign up at [000webhost.com](https://000webhost.com)
2. Upload files via FTP
3. Your site is live with a free domain

## 🐛 Troubleshooting

**Problem**: Images not showing
- Solution: Update image paths to correct URLs or use relative paths

**Problem**: Styles not loading
- Solution: Check file paths in HTML, ensure CSS file is in correct folder

**Problem**: Hamburger menu not working
- Solution: Check browser console for errors, ensure script.js is loaded
